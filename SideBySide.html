Slime Lasso! - game jam shenanigans

This game was made in a weekend for the ScoreSpace game jam! I did all the programming while the artist I worked with made the art assets and sounds. Together we made design decisions in order to refine the core game loop. I learned so much about the importance of communication between people working on different aspects of a project from this jam. It was important to us that the game felt cohesive, and to do that we regularly checked in with each other.

I'm also very proud of the lasso mechanic I made. It varies the rotational speed based on distance and time. The longer you lasso, the faster you can launch the slimes.
